# Blinkit-Sales-Dashboard
📊 Blinkit Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project presents a detailed sales analysis of Blinkit using Power BI. The dashboard provides insights into sales performance, product distribution, outlet characteristics, and customer preferences to support data-driven decision making.

📈 Key Metrics
Total Sales: $1.20M
Average Sales: $141
Number of Items: 8523
Average Rating: 3.9
📊 Dashboard Insights
🏪 Outlet Analysis
Comparison of sales across different outlet sizes (Small, Medium, High)
Sales distribution based on outlet location tiers (Tier 1, Tier 2, Tier 3)
Outlet establishment trend showing growth over time
🛒 Product Analysis
Sales breakdown by item type (Fruits, Snacks, Dairy, Household, etc.)
Identification of top-performing product categories
Analysis of product demand patterns
🥗 Fat Content Analysis
Comparison between Low Fat and Regular products
Impact of fat content on total sales
📍 Outlet Type Performance
Sales comparison across different outlet types (Grocery Store, Supermarket Types)
Metrics like average sales, number of items, and visibility
📉 Key Observations
Tier 3 outlets generate the highest sales contribution
Medium-sized outlets dominate in overall sales
Fruits and Snacks are among the top-selling categories
Low Fat products have a slightly higher contribution compared to Regular
