# Blinkit-Sales-Dashboard
# 🛒 Blinkit Sales Analytics Dashboard (Power BI)

A dynamic and interactive data visualization dashboard built to analyze Blinkit’s sales performance—focusing on outlet analysis, product trends, and customer preferences.

---

## 📌 Short Description / Purpose

The Blinkit Sales Dashboard is a Power BI report designed to analyze sales data and generate actionable business insights. It helps understand product performance, outlet distribution, and sales trends to support data-driven decision-making.

---

## 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 **Power BI Desktop** – Main platform for building interactive dashboards
• 📂 **Power Query** – Used for data cleaning and transformation
• 🧠 **DAX (Data Analysis Expressions)** – Used for calculated measures and KPIs
• 📝 **Data Modeling** – Relationships created for better analysis and filtering
• 📁 **File Formats** – `.pbix` for dashboard, `.png` for preview

---

## 📂 Data Source

The dataset used in this project contains Blinkit sales-related information including:

* Product categories (Fruits, Snacks, Dairy, etc.)
* Outlet types and sizes
* Outlet location tiers (Tier 1, Tier 2, Tier 3)
* Sales, ratings, and item details

*(Note: Dataset is for analysis and visualization purposes.)*

---

## 🚀 Features / Highlights

### 📉 Business Problem

Retail businesses like Blinkit generate large amounts of sales data, but it is difficult to quickly analyze:

* Which products perform best?
* Which outlet types generate more revenue?
* How location and size impact sales?
* What are customer preferences?

---

### 🎯 Goal of the Dashboard

To build an interactive dashboard that:

* Provides clear insights into sales performance
* Helps compare outlet types, sizes, and locations
* Identifies top-performing product categories
* Supports business decisions using data
